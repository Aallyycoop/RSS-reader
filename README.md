# RSS-reader

## Description:
RSS-reader - is a service which helps to aggregate, show and auto-update unlimited number of RSS feeds.

## System requirements:
Node.js 19.3.0 or higher

## Installation:

```
git clone git@github.com:Aallyycoop/RSS-reader.git
```
```
make install
```
```
make develop
```
```
make build
```
